## 🏔️ Sobre Red BRC

**Red BRC** es **la red colaborativa que está revolucionando Bariloche**. Somos una plataforma que conecta turistas con experiencias auténticas y comercios locales de calidad.

### 🎯 Propuesta de Valor
- **"Red viral de comercios locales + Experiencias auténticas"**
- **Flywheel perfecto**: Turistas → Comercios → Más contenido → Más turistas
- **Efecto red**: "La red más viral de Bariloche"

### 🚀 Lógica de Negocio

**El genio de esta estrategia**: Los comercios no pagan por publicidad, pagan por acceso a una audiencia ya construida. El contenido turístico es nuestro lead magnet para ambos lados del mercado.

#### Flywheel de Crecimiento:
1. **Turistas llegan** por contenido de calidad ("Descubre Bariloche")
2. **Tráfico/audiencia** atrae a comercios ("Únete a la Red")
3. **Más comercios** = más contenido = más turistas
4. **Efecto red**: "La red más viral de Bariloche"

#### Servicios Principales:
- 🎯 **Contenido Turístico**: "Qué hacer en Bariloche"
- 🍽️ **Gastronomía Local**: "Dónde comer"
- 🗺️ **Experiencias Auténticas**: "A dónde ir"
- 🤝 **B2B Conversion**: "Únete a la Red"
- 📝 **SEO + Casos de Éxito**: Blog

## 🚀 Cómo Ejecutar Localmente

### Prerrequisitos
- Python 3.x instalado
- Navegador web moderno

### Pasos de Instalación

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/gparrar0x/redbariloche.git
   cd redbariloche
   ```

2. **Iniciar servidor local**
   ```bash
   python3 -m http.server 8000
   ```

3. **Abrir en navegador**
   ```
   http://localhost:8000
   ```
